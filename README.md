## Steps
1. Check python version: 
python --version
3.11.0

2. Create a virtual environment:
python -m venv venv

3. Use the enviroment:
venv\Scripts\activate

4. Install dependencies:
pip install -r requirements.txt

3. edit app.py

4. run streamlit:
streamlit run app.py
